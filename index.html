Brick Shooter

Brick Shooter is a small and simple game made in unity. It has a retro style and a very simple concept. You Drag to shoot But be careful there is a cool down, the objective is to destroy all bricks before the time runs out. The bricks and Time limit are random allowing for relaxed gameplay which can become tense as you slowly run out of lives. I made this in 2 days but don't worry I put a lot of effort into this game making sounds and sprites. This is my first game (that I'm publishing) so please message me about any bugs at cratior#7703 on Discord.
